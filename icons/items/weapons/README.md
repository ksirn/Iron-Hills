# Iron Hills — иконки оружия

## Куда класть

Файлы должны лежать прямо здесь, имя файла — `{id}.webp` (или `{id}.png`),
где `{id}` точно совпадает с ключом из `module/constants/items-catalog.mjs` → `WEAPONS`.

Примеры:

```
icons/items/weapons/copper_sword.webp
icons/items/weapons/world_cutter.webp
icons/items/weapons/halberd.webp
icons/items/weapons/orichalcum_bow.webp
```

## Какой формат и размер

- **Формат**: `.webp` предпочтительно (меньше веса при том же качестве).
  `.png` тоже работает. `.jpg` — нежелательно (нет прозрачности).
- **Прозрачный фон**: желательно. Если AI отдаёт тёмный фон —
  можно прогнать через [remove.bg](https://remove.bg) или [Photopea](https://photopea.com).

## Какие пропорции (важно!)

Инвентарная сетка использует **прямоугольные слоты** (`gridW × gridH`).
Рекомендуемые resolution-пресеты для AI-генератора:

| `gridW×gridH` | Aspect | Resolution (AI) | Типы оружия |
|---|---|---|---|
| 1×1 | 1:1   | 1024×1024 | метательное (стопкой/веером) |
| 1×2 | 1:2   | 768×1536  | ножи, булавы 1H, цепы, чакрамы |
| 1×3 | 1:3   | 512×1536  | мечи 1H, луки, топоры 1H |
| 1×4 | 1:4   | 384×1536  | копья, двуручи, посохи, тяжёлые арбалеты |

В Midjourney: `--ar 1:3` / `--ar 1:4` / `--ar 1:2`.
В SD/Flux: задаёшь `width` × `height` напрямую.

## Ориентация

Длинное оружие должно быть **вертикальным**, остриём вверх (или влево с лёгким наклоном).
Промпт это уже подсказывает (`vertical orientation, blade pointing up`).

## Как подключить к каталогу

После того как ты сгенерируешь картинки, запусти:

```bash
node tools/apply-weapon-images.mjs
```

Скрипт пройдётся по всем файлам в этой папке и автоматически пропишет
поле `img: "systems/iron-hills-system/icons/items/weapons/{id}.webp"` в
каждую соответствующую позицию `WEAPONS` в `items-catalog.mjs`.

## Что делать с уже сгенерированными квадратными картинками

Если ты уже сгенерировал меч в формате 1024×1024 — **работать будет**, просто
картинка займёт центральную треть слота. Можно:

1. **Оставить как есть** — нормально работает.
2. **Перегенерировать** в 1:3 (рекомендую для красоты).
3. **Дополнить** — взять квадратную картинку и расширить холст вверх/вниз
   через AI-outpainting (Photoshop Generative Fill / Photopea / SD inpaint).
